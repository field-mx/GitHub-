**龙虾操作过程**  
1.开启win，双击niko.bat（确保梯子开启tun模式）  
2.自动进入Linux界面，输入账户密码  
3.打开命令行，输入“openclaw gateway run”【插入图片】  
4.打开浏览器，新建浏览页面，右上角插件开启，处于“on”代表完成配置【插入图片】  
5.可以愉快的和niko对话啦
